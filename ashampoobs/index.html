简介

Ashampoo 是德国的一家公司，公司成立于1999年，国人戏称其译名为“阿香婆”，这家公司的产品线很长，出的软件五花八门，至少也有20多款软件，包括了杀毒杀马的、防火墙的、刻录的、截图的、办公的、系统优化的、系统卸载清理、磁盘整理的、图片处理的、网络视频下载的、视频转换、音乐播放的、文件加密的等等。软件界面一般都比较华丽，但其软件并非华而不实，有很多软件越加受到国人的喜爱。
和 Nero 相比，Ashampoo 出品的刻录工具 Ashampoo Burning Studio 优势就凸显了，体积仅有 Nero 的五分之一，功能依然强大；虽然 Nero 在用户们的呼吁下推出了一款不到 40MB 的精简版 Nero Lite，毕竟是以牺牲软件特性功能为代价，并不能满足那些要求苛刻的用户，而这时 Ashampoo Burning Studio 就成为一个不错的选择。
