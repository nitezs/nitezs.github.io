Music Downloader UI 帮助

发表于2021-05-31|更新于2022-05-20|冲浪技巧

需要开启本地API之后才能使用音源2

依赖

.Net Framework 4.6.2 或以上

设置

  • 保存路径

    • 设置下载的音乐及相关文件被保存到哪里（第一次启动时的默认路径是系统的音乐文件夹）

  • 单次搜索数量

    • 设置搜索音乐时能搜索到的最大数量（只对音源1有效，音源2固定60首）

  • 下载品质

    • 标准（128000）和 较高（320000）下载的是 .mp3 格式
    • 无损（999000）下载的是 .flac 格式

  • 文件名保存格式

    • 标题 – 歌手 示例：晴天 – 周杰伦
    • 歌手 – 标题 示例：周杰伦 – 晴天

  • 路径保存格式

    • 无 示例：D:\Music\晴天 – 周杰伦.flac
    • 歌手 – 音乐 示例：D:\Music\周杰伦\晴天 – 周杰伦.flac
    • 歌手 – 专辑 – 音乐 示例：D:\Music\周杰伦\叶惠美\晴天 – 周杰伦.flac

  • 外语歌曲歌词下载类型

    • 外语、翻译、外语+翻译 （只对音源1有效）

  • 自定义音源1

    • 音源1不再依赖API

  • 自定义音源1Cookie

    • 设置音源1的用户登录Cookie（留空则使用程序内置Cookie，此功能是为了预防我的服务器不再提供接口服务，其他人可以自行搭建）

  • 自定义音源2

    • 设置音源2的接口服务器（留空则使用程序内置接口链接，此功能是为了预防我的服务器不再提供接口服务，其他人可以自行搭建）

  • 下载歌词选项

    • 在通过 “下载选中音乐” 同时下载对应歌词文件

  • 下载封面选项

    • 在通过 “下载选中音乐” 同时下载对应专辑图片

  • 启用本地API

    • 第一次启用该选项时间可能较长（大概数分钟），请耐心等待

      如果长时间加载请强制关闭程序 -> 按下 Windows+R -> 输入 %Appdata% -> 找到 MusicDownloader 文件夹 -> 删除文件夹下的文件

搜索

该文本框与音源对应，出现无法下载请切换音源。

可输入的内容有：

  1. 歌曲名
  2. 歌手名

解析歌单（playlist）

该文本框与音源对应，出现无法下载请切换音源。

可输入的内容有：

  1. 链接
  2. ID

不可以直接输入歌单名进行搜索

获取方法（以网易云音乐为例）

获取到分享链接（也可以在网页版网易云音乐中获取）：https://music.163.com/playlist?id=2399935867&userid=395076493 其中 ?id= 后面对应的就是歌单ID（2399935867） 可以直接把链接复制到歌单框中也可以只复制ID

解析专辑（album）

该文本框与音源对应，出现无法下载请切换音源。默认接口不支持下载付费专辑。

可输入的内容有：

  1. 链接
  2. ID

不可以直接输入专辑名进行搜索

获取方法（以网易云音乐为例）

获取到分享链接（也可以在网页版网易云音乐中获取）：https://music.163.com/album?id=80440087&userid=395076493 其中 ?id= 后面对应的就是专辑ID（80440087） 可以直接把链接复制到专辑框中也可以只复制ID

快捷键

在搜索列表中

  • 空格键 选中
  • Ctrl+X 下载
  • Ctrl+A 全选
  • Ctrl+R 反选
  • Ctrl+P 播放
  • Ctrl+O 暂停/继续

在音源选择框

  • Ctrl+Q 切换到下一音源

手动部署本地API

不推荐

  1. 下载所需文件
  2. 打开运行(Windows+R)
  3. 输入 %appdata% 并回车
  4. 找到 MusicDownloader 文件夹(如果没有就手动创建)
  5. 将文件解压进去
  6. 创建 INSTALL2.txt
  7. 创建 Api.txt 文件，文件内容为 3
  8. 打开 QQMusicApi 文件夹
  9. 在路径中输入 cmd 后回车
  10. 输入 npm install --registry=https://registry.npm.taobao.org 回车后等待安装
  11. 输入 npm start 后不要关闭命令窗口，打开链接 http://127.0.0.1:3300/ 测试Api部署状态，如果不能打开，在命令窗口输入 npm install -g npm 回车后再重新执行11

FAQ

  • 问：为什么我下载的音乐全是0kb？

  • 答：把你的代理关掉或者更换音质。

  • 问：怎么恢复默认背景？

  • 答：删除图片或者修改配置文件（.config）。

  • 问：下载的音乐提示无版权怎么办？

  • 答：切换音源重新搜索一遍再下载如果还不行再到设置里调整下载的音质再试一次，如果还不行就没办法了。
